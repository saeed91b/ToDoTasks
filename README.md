# ToDoTasks Repository
This is a basic .NET Web API app that performs CRUD operations allowing users to access, create, delete and edit to-do tasks.

# How to Run
1. Clone the app to your machine:
   ```
   git clone https://github.com/saeed91b/ToDoTasks.git
   ```
2. Restore the packages:
   ```
   cd ToDoList
   dotnet restore
   ```
3. Run it:
   ```
   cd API
   dotnet run
   ```
  go to http://localhost:5001/tasks to access the API.
